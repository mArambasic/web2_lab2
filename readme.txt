Popis implementiranih ranjivosti:

1. SQL ubacivanje (SQL Injection)
2. Vanjski XML entiteti (XML External Entity, XXE)
3. Nesigurna pohrana osjetljivih podataka (Sensitive Data Exposure)

Napomene:

Sve je uspješno implementirano. 
Štoviše, realizirao sam još jednu ranjivost (!) makar je zadatak tražio 2. Nadam se da zbog toga neću dobiti manje bodova, ali ovaj projekt mi se baš svidio... :)

Kratke upute kako lokalno pokrenuti i isprobati aplikaciju:

- ZIP datoteku web2.zip s izvornim kodom raspakirati u osnovnom direktoriju web servera koji podržava barem PHP4.
- Stvoriti direktorij ./web2 u kojemu se nalazi izvorni kod aplikacije.
- Otvoriti link: http://localhost/web2/
